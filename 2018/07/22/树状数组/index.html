树状数组

发表于 2018-07-22

简单但好用的数据结构

虽然它能干的事情线段树都能干。。。

但是！！！！！

它也是有优点的：

常数极小，相对线段树而言有更优的时间，而且不需要4倍数组，这就意味着内存也不是问题。而且写起来简单方便，在考场上比较适用。所以对于区间问题，考虑它可以在线段树之前。

用处

可以维护前缀和以及区间的最值，相当于线段树的题型一。

不过与线段树相同的是，它同样需要离散化，不过这不是重点。

主要用处

  1. 前缀和（单点修改）
  2. 区间查询大与等于它的数有多少，有机地配合逆序对
  3. 区间最值（单点修改）

只要知道$C[i]$数组表示$i…i-lowbit(i)+1$就好了。

下面举几个例题，树状数组的应用。

Ping pong（HDU2492）

题目描述

给定一个长度为N的序列，序列中的元素都不相同，有多少个下标三元组$(i＜j＜k)$满足$A[i]＜A[j]＜A[k]$或者$A[i]＞A[j]＞A[k]$

思路

相当求一个数左右两边有多少的数大于它和小于它的数，那么我们只要正反扫两边数组，在用树状数组维护就好了，最后再计算答案。

magic balls（HDU5125）

题目描述

给定两个长度为N的序列a和b，可以把ai和bi进行交换，最多进行M次 求如何交换，序列a的最长上升子序列最长。

思路

离散化+dp+树状数组。（用处三）

本来想用线段树维护，然后就MLE了。。8倍数组卡内存好骚

这是就体现出了树状数组的优越性，不需要太多的内存就可以完成维护最大值。（虽然这题也可以用纯nlong的LIS写法。。）

不过需要注意的是Update的时候需要将A、B数组同时更新（这个点卡了一个晚上啊啊啊啊啊），否则后面的LIS不会是最优的（其实就和nlogn写法差不多）

Tunnel Warfare（HDU1540）

题目描述

有几个村庄，D a表示a村庄被摧毁，R表示修复上一个村庄，Q a表示包含a的连续村庄有多少个，如果a被摧毁则输出0。

思路

前缀和的灵活应用（虽然可以用线段树写。。）

二分+树状数组，我们只要满足一个区间全是1就好了，也就是：

$sum[i]-sum[j-1]=i-j+1$

显然具备单调性，然后就可以二分了~~

注意一个村庄可以被摧毁好多遍，但修一下就修好了（卡了一个早上的点。。。），再开一个数组判。
